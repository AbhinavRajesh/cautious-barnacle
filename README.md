# Board. (Listed)

This is the source code for the application running at: https://listed-assignment-ar.vercel.app/
Technologies used:
- ReactJS
- NextJS
- TailwindCSS
- TypeScript
- Next-Auth
- ChartJS

## Steps to setup

1. Clone the repository

   ```bash
   git clone https://github.com/AbhinavRajesh/cautious-barnacle.git
   ```

2. Change the working directory

   ```bash
   cd cautious-barnacle
   ```

3. Install dependencies

   ```bash
   yarn
   ```

4. Run the dev server

   ```bash
   yarn dev
   ```

Open [localhost:3000](http://localhost:3000/) to see the app. ðŸŽ‰
